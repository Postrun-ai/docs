The Instagram media_type does not match the post_type.

## When it happens

An Instagram variant’s declared `settings.media_type` contradicts its `post_type` — a single image must be `IMAGE`, a carousel `CAROUSEL`, a reel `REELS`. Rejected here rather than silently overridden. Surfaced in the `not_publishable` `errors[]`.

## How to resolve

Set `media_type` to the value the `post_type` implies (or omit it where allowed) and resubmit.
