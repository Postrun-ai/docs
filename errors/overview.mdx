title: "Error codes"
description: "Every error Postrun can return, with the condition that triggers it and how to resolve it."
---

Postrun returns a **closed set** of machine-readable error codes. Branch on the
`code` field, never the message. Every error body's `type` links to that code's
page here. Each endpoint's reference lists the finite set it can return.

There are two channels, same vocabulary:

- **Transport errors** ride a non-2xx [RFC 9457](https://www.rfc-editor.org/rfc/rfc9457) problem body (`{ type, title, status, code, detail, request_id }`).
- **Domain verdicts** ride *inside a 200* (e.g. `media.per_platform[].errors[]`, a post variant's `error`) — the validation outcomes you resolve per item.

## Transport errors

| Code | Status | Retryable | Meaning |
| --- | --- | --- | --- |
| [`unauthorized`](/errors/unauthorized) | 401 | No | The request carried no valid Postrun credential. |
| [`forbidden`](/errors/forbidden) | 403 | No | The credential is valid but not permitted to do this. |
| [`not_found`](/errors/not_found) | 404 | No | The requested resource does not exist for this account. |
| [`conflict`](/errors/conflict) | 409 | No | The request conflicts with the resource’s current state. |
| [`validation_failed`](/errors/validation_failed) | 422 | No | The request body or parameters failed schema validation. |
| [`rate_limited`](/errors/rate_limited) | 429 | Yes | Too many requests for this account in the current window. |
| [`internal_error`](/errors/internal_error) | 500 | Yes | An unexpected server-side failure. |
| [`idempotency_key_invalid`](/errors/idempotency_key_invalid) | 400 | No | The Idempotency-Key header is malformed. |
| [`idempotency_key_reused`](/errors/idempotency_key_reused) | 422 | No | This Idempotency-Key was reused with a different payload. |
| [`idempotency_request_in_progress`](/errors/idempotency_request_in_progress) | 409 | Yes | A request with this Idempotency-Key is still running. |
| [`source_url_unsupported`](/errors/source_url_unsupported) | 422 | No | Server-side fetch of a `source_url` is not supported in v1. |
| [`account_not_available`](/errors/account_not_available) | 422 | No | The chosen account is not selectable on this connection. |
| [`connection_reauth_required`](/errors/connection_reauth_required) | 409 | No | The connection must be reconnected before selecting an account. |
| [`connection_not_pending`](/errors/connection_not_pending) | 409 | No | This connection already has an account selected. |
| [`not_implemented`](/errors/not_implemented) | 501 | No | This capability is not yet available for the platform. |
| [`media_processing`](/errors/media_processing) | 409 | Yes | The media asset is still being processed. |
| [`not_publishable`](/errors/not_publishable) | 422 | No | One or more variants are not ready to publish. |
