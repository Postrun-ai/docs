A reel-only field was set on a non-reel post.

## When it happens

An Instagram variant whose `post_type` is not `reel` set a reel-only setting (a field valid only on reels). Surfaced in the `not_publishable` `errors[]`.

## How to resolve

Remove the reel-only field (the error path names it), or change `post_type` to `reel` if a reel is what you intended, then resubmit.
