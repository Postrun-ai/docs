Instagram throttled the request.

## When it happens

The Graph API returned a rate-limit error code (4 / 17 / 32 / 613) — application- or user-level throttling. Returned as the Instagram variant’s `outcome.error`.

## How to resolve

Wait and republish later; spread Instagram publishes out. Transient — the same request can succeed once the limit resets.

## Reference

| Field | Value |
| --- | --- |
| Code | `instagram_rate_limited` |
| Channel | Domain verdict — returned inside a 200 response |
| Retryable | Yes — the same request may succeed if retried. |
