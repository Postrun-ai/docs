The Facebook Page token is invalid or lacks permission.

## When it happens

The Graph API returned an `OAuthException` (or code 190) — an invalid/expired Page token or a missing permission. Returned as the Facebook variant’s `outcome.error`.

## How to resolve

Reconnect the Facebook Page to refresh the token and grant the needed permissions, then republish. Show your user: "Please reconnect your Facebook Page."

## Reference

| Field | Value |
| --- | --- |
| Code | `facebook_not_authorized` |
| Channel | Domain verdict — returned inside a 200 response |
| Retryable | No — the request must change before it can succeed. |
