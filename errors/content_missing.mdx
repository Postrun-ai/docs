The variant has no publishable content.

## When it happens

A publish variant carries nothing to post — e.g. an X post with no text, media, poll, quote, or card; a text-only LinkedIn post with blank commentary; a Facebook text post with no message or link. Surfaced in the `not_publishable` `errors[]`.

## How to resolve

Add body text or the appropriate content object (media/poll/quote/card/link) for the platform, then resubmit. This will not pass until the variant has content.

## Reference

| Field | Value |
| --- | --- |
| Code | `content_missing` |
| Channel | Domain verdict — returned inside a 200 response |
| Retryable | No — the request must change before it can succeed. |
